GET HELIX

Installing on Android

Three steps, about a minute, one time only.

1

Download the file

Tap "Download for Android" above. The file saves to your Downloads folder.

2

Allow this install

Android will ask permission the first time. Tap Settings in that prompt, then turn on "Allow from this source."

3

Install and open

Go back, tap Install, then open HELIX from your home screen or app drawer.

On iPhone, Chromebook, or desktop? Open the web app instead — it installs straight from your browser, no file needed.
